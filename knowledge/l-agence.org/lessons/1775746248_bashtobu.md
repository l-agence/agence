## Lesson

When extracting bash data logic into Bun TS, the clean separation is:

- **Bun gets**: tables, lookups, config parsing, env detection, structured output (JSON, shell exports)
- **Bash keeps**: curl calls, TTY formatting, system prompt building, interactive I/O

### Delegation Pattern

```bash
# In router_load_config():
_bun_exports=$(bun run lib/router.ts resolve-route ... 2>/dev/null)
if [[ $? -eq 0 && -n "$_bun_exports" ]]; then
  eval "$_bun_exports"   # safe: router.ts sanitizes output
  return 0
fi
# fallback to bash logic
```
