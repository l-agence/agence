# Agence ROADMAP.md

## Vision
Agence is a modular, auditable, and security-focused agentic operating system for distributed, multi-agent, multi-org AI collaboration. It is designed for provenance, auditability, and federated cognition at a level unmatched by vendor LLMs, open-source agentic tools, or current AI OS projects.

## Key Differentiators
- **Universal Routing & State Model:** Canonical, extensible routing and state prefixes for agents, orgs, security, and context.
- **Hermetic vs. Synthetic Knowledge:** Explicit split between immutable, provenance-tracked objectcode and derived, shared synthetic knowledge.
- **COGNOS Cognition Model:** Clear separation of agent state, session, context, and provenance.
- **Objectcode vs. RAG/Semantic:** Immutable, git-tracked code/metadata separated from mutable, derived knowledge.
- **Distributed Working Memory (DWM):** Tiered (private, shared, federated) memory for knowledge routing.
- **Security Labelling:** First-class security labels enforced in routing and state.
- **Swarm State & Scheduling:** Distributed, idempotent swarm/project state, automatic agent scheduling, heatmap/bottleneck detection, and swarm-merge with locking.
- **Cost Efficiency:** Intelligent, cost-aware agent/model selection and resource allocation.
- **Session Persistence:** Persistent, tiered session capture to prevent hallucination and context loss.
- **Dewey-like Formtag KB:** Hierarchical, provenance-rich, human-readable knowledgebase.
- **CLI & Prompt Integration:** Works seamlessly from shell, scripts, and interactive prompts.
- **Modular Integration:** Designed for smooth integration with Aider, Cursor, Ollama, Ralph, MIT Peers, and more.
- **Tiered Knowledge Dissemination:** Explicit hermetic (private), synthetic (shared), and federated (swarm/global) tiers.

## Unique Constructs
- Universal routing and state model
- Hermetic/synthetic knowledge split
- Dewey-like, provenance-rich knowledgebase
- Explicit cognition and session models
- Distributed, idempotent swarm/project state
- Security labelling and auditability
- Intelligent, cost-aware agent scheduling
- CLI, prompt, and modular integration
- Future-proofed for ledgers and enterprise integration

## Future Features
- **.ailedger & Allegiance Ledger:** Immutable, auditable ledgers for agent actions, allegiance, and provenance.
- **JIRA Integration:** Seamless project/task sync with enterprise workflows.
