## 📌 Help Subcommand Expansion (added 2026-04-01)

`agence help <subcommand>` must work universally across two categories:

### Category 1 — Agence-native (delegate to tool)
| Command | Delegates to |
|---------|-------------|
| `agence help swarm` | `swarm help` |
| `agence help aido` | `aido help` |
| `agence help session` | `aisession help` |
| `agence help [ibash\|aibash\|ishell\|aishell]` | respective tool help |
| `agence help symbols` | `^symbols` output |
| `agence help agents` | list all personas (@ralph, @sonya, @aider, @copilot...) |
| `agence help [lesson\|log\|plan\|todo\|fault\|issue\|task\|job]` | knowledge command help |

### Category 2 — External tools (agence lists ITS subcommands for category)
| Command | Shows |
|---------|-------|
| `agence help git` | agence git subcommands (from commands.json tiers) |
| `agence help gh` | agence GitHub CLI subcommands |
| `agence help [terraform\|tf]` | agence terraform subcommands |
| `agence help [aws\|az\|azure\|gcp]` | cloud provider subcommands |
| `agence help [jf\|jfrog]` | JFrog subcommands |
| `agence help docker` | agence docker subcommands |
| `agence help [k8s\|kubectl]` | kubernetes subcommands |
| `agence help jira` | Jira integration subcommands |
| `agence help tempo` | Tempo time-tracking subcommands |
| `agence help confluence` | Confluence integration subcommands |
| *(extensible: new integrations added here)* | |

**Pattern**: External tools = agence owns the help text (no wrapping). Native tools = delegate to tool.

## 🔧 Technical Requirements

### **Routing Inheritance**
```bash
^plan list
  → Check @current_context (default: @)
  → Fallback: synthetic/@/plans/
  
^plan list --org acme.tld
  → Explicit: synthetic/@acme.tld/plans/
  → Fallback: synthetic/@/plans/
```

### **Shell Completion**
```bash
^plan [list|show|add] [--org NAME]
^todo [list|show|add]
^lesson [list|show|add] [--org NAME]
# etc.
```

### **Local-Only Enforcement** (hermetic/@)
```bash
# ^todo is ALWAYS local (never committed)
^todo list  → hermetic/@/todos/  (only)
^todo add "Refactor lib.sh"  → ~/.agence/hermetic/@/todos/
```
