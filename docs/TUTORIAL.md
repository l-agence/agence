# Agence Getting Started Guide

Welcome to the Agence getting started guide! This document aims to help new users familiarize themselves with command routing and provides practical examples of Agence's tool-agnostic approach.

## Command Routing

Command routing in Agence allows you to easily manage various commands and their execution flows. Understanding how to route commands effectively will enhance your productivity and enable you to leverage Agence's capabilities more efficiently.

### Basic Command Structure

Every command follows a basic structure. Here’s a simple example:

```bash
command_name --option value
```

### Example Usage

Here's how you can use the command routing feature in practice:

```bash
# Fetching user information
fetch_user --id 12345

# Sending a request
send_request --url https://example.com/api
```

## Tool-Agnostic Approach

Agence emphasizes a tool-agnostic approach, which means that it can work with various tools without being tied to a specific technology or platform. This flexibility allows you to choose the best tool for your needs without restriction.

### Benefits of a Tool-Agnostic Approach
- **Flexibility**: You can integrate multiple tools as needed.
- **Scalability**: Easily scale your operations without changing the underlying framework.
- **Cost-effectiveness**: Utilize existing tools instead of investing in new ones.
